# ZeroCut Director Notes

面向 AI 视频导演的知识库仓库，用于沉淀可复用的提示词板与视觉特效模板。

## Vibe Directing

`Vibe Directing` 可以理解为一种以导演感受、镜头氛围、节奏判断和视觉意图为核心的 AI 视频创作方法。

在这个方法里，创作者只需要用自然语言提出“想做什么”和“目的是什么”，而导演提示词板负责把下面这些关键内容拆解清楚：

- 这场戏的情绪和氛围是什么
- 镜头应该怎么看、怎么切、怎么推进
- 角色应该如何表演、如何反应、如何调度
- 画面中的空间关系、动作节奏和视觉重点是什么

也就是说，AI 视频创作不是让创作者自己一次性写完所有细节，而是先提出自然语言需求，再选择合适的导演提示词板作为参考图，把镜头、表演、动作、节奏和空间关系整理清楚，然后交给 AI 去解决具体生成。

这个仓库里的 `notes/` 和 `effects/`，就是为 `Vibe Directing` 服务的参考图集合。实际使用时，可以直接选择仓库里的提示词板作为参考图，再配合用户提供的角色卡、环境卡来保持主体和场景一致性，同时补上一句自然语言描述的“想做什么 / 目的是什么”，剩下的镜头拆解、风格延展和画面实现交给 AI 去完成。

## 目录结构

```text
.
├── demos/          # 示例视频，展示提示板或特效模板的应用效果
├── effects/        # 视觉特效模板、结尾包装、镜头转场参考
├── LICENSE         # 自定义授权协议
├── notes/          # 通用提示词板、导演笔记、表演与镜头知识卡
├── metadata.json   # 当前知识库条目的结构化索引
└── README.md
```

## 收录原则

- `notes/`：放通用的提示词板、导演知识卡、镜头语言和表演控制参考。
- `effects/`：放视觉特效模板、镜头包装模板、转场和结尾样式参考。
- `metadata.json`：维护每个条目的标题、说明、适用场景、标签，方便后续检索与程序化读取。

## 使用方式

- 浏览素材时，优先查看 `metadata.json`，快速了解每个提示词板或特效模板的用途。
- 实际创作时，直接选择仓库里的 `notes/` 或 `effects/` 作为参考图。
- 再配合用户的角色卡、环境卡，确保人物形象、场景设定和风格一致。
- 最后补上一句自然语言需求，说明“想做什么”以及“目的是什么”。
- 其余镜头拆解、动作组织、氛围延展和画面实现，交给 AI 在参考图基础上继续完成。
- 新增 `notes/` 或 `effects/` 中的素材后，同步更新 `metadata.json`。
- 如果素材适合长期复用，建议在说明中补充：
  - 主题或用途
  - 适合的镜头或场景
  - 可直接复用的提示词方向
  - 检索标签
