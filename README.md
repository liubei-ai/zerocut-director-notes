# ZeroCut Director Notes

面向 AI 视频导演的知识库仓库，用于沉淀可复用的提示词板与视觉特效模板。

## 目录结构

```text
.
├── demos/          # 示例视频，展示提示板或特效模板的应用效果
├── effects/        # 视觉特效模板、结尾包装、镜头转场参考
├── LICENSE         # 自定义授权协议
├── notes/          # 通用提示词板、导演笔记、表演与镜头知识卡
├── metadata.json   # 当前知识库条目的结构化索引
└── README.md
```

## 收录原则

- `notes/`：放通用的提示词板、导演知识卡、镜头语言和表演控制参考。
- `effects/`：放视觉特效模板、镜头包装模板、转场和结尾样式参考。
- `metadata.json`：维护每个条目的标题、说明、适用场景、标签，方便后续检索与程序化读取。

- `notes/死神来了.png`

  - 多人追逐与挑逗切换通用示意板。
  - 适合一人追多人、目标切换、猫鼠游戏式追逐场景。

- `notes/宏大战争场面.png`
  - 宏大战争场面导演提示板。

- `notes/微表情.png`
  - 导演微表情提示词板。
  - 适合角色近景、特写、情绪对峙场景。

### 特效模板

- `effects/画面入古书.png`
  - 任意画面压入古书并合上的通用特效模板。
  - 适合片头片尾、章节封存、知识记录感和古籍风包装场景。
  - 强调画面书页化、古书收纳、合书定格与封面呈现。

- `effects/递归循环世界.png`
  - 从局部细节递进到宇宙再回到起点的递归循环特效模板。
  - 适合梦境、幻觉、意识空间、宇宙叙事和无限轮回感场景。
  - 强调尺度切换、光点汇聚、闭环结构与循环回归。

- `effects/历史剧终.png`
  - 历史题材结尾包装模板。
  - 核心效果是将全景画面压入书页，最终合书成封面。
  - 适合历史剧、史诗感短片、章节结束镜头。

## 使用方式

- 浏览素材时，优先查看 `metadata.json`，快速了解条目含义和用途。
- 新增 `notes/` 或 `effects/` 中的素材后，同步更新 `metadata.json`。
- 如果素材适合长期复用，建议在说明中补充：
  - 主题或用途
  - 适合的镜头或场景
  - 可直接复用的提示词方向
  - 检索标签
